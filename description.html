Rain Quest

An old school text parser adventure game in the style of the AGI Sierra classic games like Kings Quest and Space Quest

You play a guy named Stan the man, who has just been dumped by his girlfriend. Although it is bad news, you can now go to a convention based on your favorite game Face Quest.

Your girlfriend left you for dead so now you must make the trek back to your house, get the convention tickets, and go to the convention.

* This game is under development so if you find any bugs let me know!

** Part 1 is almost complete. There will be 3 parts in total.

Movement

Direction Action
Left Arrow This will move you to the left.
Right Arrow This will move you to the right.
Up Arrow This will move you up.
Down Arrow This will move you down.

Text Parser Commands

These are the basic commands you can type. However there are many more commands available in the game.

Command Action
Look This will give a description based on what you want to look at.
Take This will take an item you find in the game.
Use You can use certain items you find in the game to progress even further.
Talk You can talk to certain characters to have conversations and get clues.
Mess With You have a handy dandy screwdriver kit that you can mess with objects.
Throw You will be throwing many objects in this game. Maybe even a frisbee.
Help This will give a brief description and give a hint on what to do.
Quit This will quit the game.

Hot Keys Commands

Hot Key Action
F1 Shows help menu
F3 Retypes your last command
F4 Save Quest Menu
F5 Save Game
F7 Restore Game
F9 Restarts the Game
Tab Show Inventory
Escape Show Top Menu Bar
